## Setup Instructions

To host this site on GitHub Pages:

1. **Create a GitHub Repository:**
   - Go to GitHub and create a new repository named `yourusername.github.io` (replace `yourusername` with your actual GitHub username).
   - Make sure it's public (or private if you have GitHub Pro).

2. **Clone the Repository:**
   - Clone the newly created repository to your local machine:
     ```
     git clone https://github.com/yourusername/yourusername.github.io.git
     cd yourusername.github.io
     ```

3. **Add the Site Files:**
   - Copy the `index.html` file from this repository into your cloned repository.

4. **Commit and Push:**
   - Add, commit, and push the files:
     ```
     git add .
     git commit -m "Initial commit: Add profile page"
     git push origin main
     ```

5. **Enable GitHub Pages:**
   - Go to your repository on GitHub.
   - Click on "Settings" tab.
   - Scroll down to "Pages" section.
   - Under "Source", select "Deploy from a branch".
   - Choose "main" branch and "/ (root)" folder.
   - Click "Save".

6. **Access Your Site:**
   - Your site will be available at `https://yourusername.github.io` after a few minutes.
   - If you made changes, they should be reflected automatically on the next push.
